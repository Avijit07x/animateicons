`@animateicons/mcp` is a <Link href="https://modelcontextprotocol.io" target="_blank" className="underline underline-offset-4">Model Context Protocol</Link> server. Connect it to an AI coding agent - Claude Code, Cursor, and others - and the agent can search the AnimateIcons catalog and drop animated icons straight into your project, no copy-pasting.

<div className="mt-10 space-y-14">

<section id="tools" className="scroll-mt-20 space-y-4">

## What it can do

Once connected, your agent gets four tools:

- `search_icons` - fuzzy-search the catalog by name, keyword, or category.
- `get_icon` - return an icon's component source plus a ready-to-paste import (read-only).
- `add_icon` - write an icon's component file into your project (default `components/icons/`).
- `list_libraries` - list the libraries and how many icons each has.

## Requirements

Icons written by `add_icon` import `cn` from `@/lib/utils` (the shadcn convention) and require the <Link href="https://motion.dev" target="_blank" className="underline underline-offset-4">motion</Link> package at runtime.

Prefer the terminal? The <Link href="/icons/docs" className="underline underline-offset-4">CLI</Link> does the same thing with `npx animateicons add`.
